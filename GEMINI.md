# Directory Overview

This directory contains an Angular project named "Task Tracker". It is a modern web application built with the Angular framework and TypeScript.

# Key Files

The primary project files are located in the `Task-Tracker` subdirectory:
*   `angular.json`: Configuration file for Angular CLI.
*   `package.json`: Defines project metadata and manages dependencies and scripts.
*   `tsconfig.json`: TypeScript configuration file.
*   `src/`: Contains the application's source code.
*   `public/`: Likely contains static assets or the build output.

# Usage

This project is intended for development and deployment as a web application.

## Building and Running

To set up and run the project:

1.  Navigate to the `Task-Tracker` directory:
    ```bash
    cd Task-Tracker
    ```
2.  Install the project dependencies:
    ```bash
    npm install
    ```
3.  To start the development server:
    ```bash
    npm start
    ```
    The application will typically be accessible at `http://localhost:4200/`.
4.  To build the project for production:
    ```bash
    npm run build
    ```
    The build artifacts will be stored in the `dist/` directory.
5.  To run the unit tests:
    ```bash
    npm test
    ```
